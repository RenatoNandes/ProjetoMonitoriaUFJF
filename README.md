## Projeto referente à prova de implementação para Monitoria de Orientação a Objetos (POO) - UFJF

A ideia do projeto consiste num acervo de livros em que se pode pegar e devolver livros emprestados, 
possuindo diferentes níveis de acesso e respeitando/usando os conceitos de POO 
(Como encapsulamento, herança, polimorfismo, classes abstratas, interfaces e coleções)
